NTD 252

你沒有更好的命運 NT252

「並非依照創作的線性時序，而是內心的狀態序列。」

《你沒有更好的命運》分成四輯，
分別為「成為影子」、「末日遠行」、「更好的命運」、和「有神繁生」。
這四輯分別象徵著自己四種不同的創作歷程，各自有其企圖與嘗試。
輯一主要為情詩，以對自己呢哺的方武作抒情練習；輯二企圖用詩反映社會上的時事群像;
輯三最親近現實，為自覺、情感與創作狀態的摹寫；輯四則是對世界、神話、可名或不可名的神祇的叩問。

你沒有更好的命運

《你沒有更好的命運》分成四輯，分別為「成為影子」、「末日遠行」、「更好的命運」、和「有神繁生」。這四輯分別象徵著自己四種不同的創作歷程，各自有其企圖與嘗試，引領讀者走進內心深刻卻模糊的疆界。 

光天化日

教我們痛、令我們不寐、讓我們看見不存在的幸福。 繼《你沒有更好的命運》後，任明信備受期待的第二本詩集，書寫愛戀中情感群像、從關係中的相遇到分離、自我內部的辯證、對世界最純粹的想望、對生命的真實呼喊。
